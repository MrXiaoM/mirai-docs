[PRs welcome](https://github.com/MrXiaoM/mirai-docs/pulls)

PR 标题尽可能简述你作出的变动，语言仅接受中文和英文，commit message 同理。

中文和西文（包括但不限于英文和阿拉伯数字）之间请用空格隔开，西文与中文标点间不需要隔开，示例：  
> 昨天 Tom 给我提交了一份 Pull Request，我还没有来得及看。mirai 2.15.0 好像更新了。
